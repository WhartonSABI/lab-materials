# 2026 Labs: Submission Instructions

For each lab, start from the file in `starter-code/` and make your own copy in `submissions/`.

## How to submit

1. Find the lab starter file in `2026/labs/starter-code/`.
2. Copy it into `2026/labs/submissions/`.
3. Rename your submission file using this format:
   - `<lab-number>_<first-name>.R`

Example:
- `01_ahana.R`

So for Lab 1, Ahana would submit:
- `2026/labs/submissions/01_ahana.R`

4. Commit and push your submission file to GitHub to submit.

## Notes

- Keep the lab number zero-padded (for example: `01`, `02`, `03`).
- Use your first name in lowercase.
- Submit only your own copied file in `submissions/`.
